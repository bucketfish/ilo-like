kala pona Tonyu made this website using Google's TeachableMachine.

the data used to train this is from the following fonts: FairfaxPona, linja waso lili, nasin nanpa, nasin sitelen pu, nishiki teki, and sitelen seli kiwen. you can find the fonts here. it also uses these handwritten svgs.

you can find the source code on Github.

the model used can be found on TeachableMachine.

the font used to display sitelen pona on this page is linja sike, and the font used to display latin characters is Roboto. the font 'nasin nanpa' is used to display the alternative epiku glyph.
